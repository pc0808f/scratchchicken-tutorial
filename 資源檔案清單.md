## 🚀 部署步驟摘要

1. **建立 Repository**：在 GitHub 創建公開專案
2. **上傳檔案**：將所有檔案推送到 main 分支
3. **啟用 Pages**：在設定中開啟 GitHub Pages
4. **等待部署**：約 5-10 分鐘完成自動部署
5. **訪問網站**：透過 `用戶名.github.io/專案名` 訪問
