# 💩 第六週:自己的便便自己清
{: .fs-9 }

電子雞會吃、會玩、會移動了,但還少了一個真實的生活細節──牠會大便!這週我們要學習進階的「分身系統」,讓電子雞定時產生便便,玩家要記得清理,否則環境會變髒喔!

## 🎬 本週遊戲展示

### 💩 便便系統玩法

**真實的照護體驗**:讓電子雞的生活更加完整!

#### 遊戲機制:
1. **定時產生便便** ⏰
   - 每隔一段時間自動產生便便
   - 便便出現在隨機位置
   - 最多同時存在 5 個便便

2. **點擊清理便便** 🧹
   - 滑鼠點擊便便即可清除
   - 清理後清潔度恢復
   - 清理音效回饋

3. **清潔度影響** 📊
   - 便便越多 → 清潔度越低
   - 清潔度低 → 電子雞會不開心
   - 清潔度影響電子雞心情

4. **計時器系統** ⏱️
   - 使用內建計時器
   - 精確控制產生時間
   - 學習時間管理概念

## 📚 學習內容

### 🎯 第一部分:認識分身系統

#### 1.1 什麼是分身?

**生活中的分身概念**:
- 🎈 **氣球**:一次放飛很多個氣球
- 🌧️ **雨滴**:天空中同時有很多雨滴
- ⭐ **星星**:夜空中有數不清的星星
- 🎆 **煙火**:煙火綻放時產生許多火花

**在 Scratch 中**:
- 一個角色可以有很多「分身」
- 每個分身都是「複製品」
- 分身可以獨立移動、改變
- 分身可以被刪除

#### 1.2 本體 vs 分身

| 特性 | 本體 | 分身 |
|------|------|------|
| 數量 | 只有 1 個 | 可以有很多個 |
| 建立方式 | 從角色庫新增 | 用程式建立 |
| 刪除方式 | 手動刪除角色 | 用「刪除此分身」 |
| 程式執行 | 執行本體的程式 | 執行「當分身產生」的程式 |

#### 1.3 分身系統的積木

**重要積木認識**:

```scratch
建立自己的分身         ← 建立一個新分身
當分身產生             ← 分身產生時執行這段程式
刪除此分身             ← 刪除這個分身
```

**💡 重要概念**:
- 「建立自己的分身」寫在**本體**的程式中
- 「當分身產生」是分身的**專屬事件**
- 「刪除此分身」只能刪除分身,不能刪本體

### 🎯 第二部分:製作便便角色

#### 2.1 準備便便角色

**步驟 1:新增便便角色**
1. 點擊「選擇一個角色」
2. 搜尋「poop」或自己畫一個
3. 調整大小(建議縮小到 30% 左右)

**步驟 2:設定初始位置**
```scratch
當綠旗被點擊
隱藏                    ← 本體一開始要隱藏!
定位到 x:(0) y:(0)
```

{: .highlight }
> **💡 為什麼本體要隱藏?**
>
> 因為我們只需要「分身」出現在舞台上,本體不需要顯示!
> 本體的工作是「建立分身」,分身才是真正的便便!

#### 2.2 建立便便分身(電子雞的程式)

**在電子雞角色中新增**:

```scratch
當綠旗被點擊
重新計時                ← 啟動計時器
重複無限次
  如果 <(計時器) > [20]> 那麼    ← 每 20 秒產生便便
    廣播訊息 [產生便便 v]
    重新計時
  結束
結束
```

**在便便角色中新增**:

```scratch
當收到訊息 [產生便便 v]
建立自己的分身          ← 建立一個便便分身
```

#### 2.3 分身出現時的設定

**便便角色的分身程式**:

```scratch
當分身產生
顯示                    ← 分身要顯示出來
定位到 x:(隨機取數(-200)到(200)) y:(隨機取數(-150)到(150))
```

**測試看看**:
- 點擊綠旗
- 等待 20 秒
- 應該會看到便便出現在隨機位置!

### 🎯 第三部分:清理便便系統

#### 3.1 點擊清理便便

**便便角色新增程式**:

```scratch
當分身產生
顯示
定位到 x:(隨機取數(-200)到(200)) y:(隨機取數(-150)到(150))
重複無限次
  如果 <碰到 [鼠標 v] ?> 那麼
    如果 <滑鼠鍵被按下?> 那麼
      刪除此分身         ← 點擊後刪除這個便便
    結束
  結束
結束
```

**測試看看**:
- 等待便便出現
- 點擊便便
- 便便應該消失!

#### 3.2 建立清潔度變數

**步驟**:
1. 建立變數「清潔度」
2. 初始值設為 100
3. 便便出現時減少清潔度
4. 清理便便時恢復清潔度

**電子雞的初始化程式(新增)**:

```scratch
當綠旗被點擊
定位到 x:(0) y:(0)
變數 [飢餓度 v] 設為 [80]
變數 [快樂度 v] 設為 [80]
變數 [清潔度 v] 設為 [100]   ← 新增!
```

#### 3.3 便便影響清潔度

**便便角色修改程式**:

```scratch
當分身產生
顯示
定位到 x:(隨機取數(-200)到(200)) y:(隨機取數(-150)到(150))
變數 [清潔度 v] 改變 [-10]     ← 新增!便便出現減少清潔度

重複無限次
  如果 <碰到 [鼠標 v] ?> 那麼
    如果 <滑鼠鍵被按下?> 那麼
      變數 [清潔度 v] 改變 [10]  ← 新增!清理恢復清潔度
      刪除此分身
    結束
  結束
結束
```

### 🎯 第四部分:進階功能與優化

#### 4.1 限制便便數量

**問題**:如果玩家不清理,便便會越來越多!

**解決方法**:建立「便便數量」變數

**便便角色修改**:

```scratch
當收到訊息 [產生便便 v]
如果 <(便便數量) < [5]> 那麼    ← 最多 5 個便便
  建立自己的分身
  變數 [便便數量 v] 改變 [1]
結束
```

```scratch
當分身產生
顯示
定位到 x:(隨機取數(-200)到(200)) y:(隨機取數(-150)到(150))
變數 [清潔度 v] 改變 [-10]

重複無限次
  如果 <碰到 [鼠標 v] ?> 那麼
    如果 <滑鼠鍵被按下?> 那麼
      變數 [清潔度 v] 改變 [10]
      變數 [便便數量 v] 改變 [-1]  ← 新增!刪除時減少計數
      刪除此分身
    結束
  結束
結束
```

#### 4.2 清潔度影響電子雞心情

**電子雞新增狀態判斷**:

```scratch
當綠旗被點擊
重複無限次
  如果 <<(飢餓度) < [30]> 且 <(清潔度) < [30]>> 那麼
    造型換成 [生氣 v]
    說出 [又餓又髒,我生氣了!] 持續 (2) 秒
  結束

  如果 <(清潔度) < [30]> 那麼
    造型換成 [傷心 v]
    說出 [好髒喔,快幫我清理!] 持續 (2) 秒
  結束

  等待 (3) 秒
結束
```

#### 4.3 調整產生頻率(選做)

**可以調整的參數**:
- 產生時間間隔(目前 20 秒)
- 清潔度影響程度(目前 -10)
- 清理恢復程度(目前 +10)
- 最大便便數量(目前 5 個)

**建議設定**:
```
簡單模式: 30 秒產生,-5 清潔度,最多 3 個
普通模式: 20 秒產生,-10 清潔度,最多 5 個
困難模式: 10 秒產生,-15 清潔度,最多 7 個
```

### 🌟 高級挑戰

   - 普通便便:清理 +10

   - 需要點擊「掃把」角色
   - 再點擊便便才能清理

3. **清潔度影響移動**:
   - 清潔度低時,電子雞移動變慢

## 📝 重點複習

### 🔑 分身系統核心概念
1. **本體只有一個,分身可以很多個**
2. **「建立自己的分身」在本體程式中**
3. **「當分身產生」是分身的啟動事件**
4. **「刪除此分身」用來清除分身**
5. **本體通常要隱藏,只顯示分身**

### 🔑 計時器應用要點
1. **「重新計時」清零計時器**
2. **「計時器」變數顯示經過的秒數**
3. **用條件判斷計時器數值**
4. **達到設定時間就執行動作**

### 🔑 程式設計重點
1. **變數管理**:同時追蹤多個數值
2. **事件觸發**:使用廣播訊息溝通
3. **隨機定位**:讓遊戲更有變化
4. **數值平衡**:調整合理的遊戲難度
