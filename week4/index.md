## 🎬 本週遊戲展示

### 🎯 我們要做什麼遊戲？

**拍拍樂互動遊戲**：一個會根據心情做出不同反應的聰明電子雞！

#### 遊戲玩法：
1. **偵測滑鼠接近** 👋
   - 滑鼠靠近電子雞 → 牠會眨眼打招呼
   - 滑鼠移開 → 牠會繼續做自己的事

2. **點擊增加快樂度** 😊
   - 點擊電子雞 → 快樂度增加
   - 快樂度低時 → 牠會說「好無聊喔」
   - 快樂度高時 → 牠會說「我超開心！」

3. **聰明判斷狀態** 🧠
   - 又餓又不開心 → 顯示「傷心」表情
   - 飽飽的又開心 → 顯示「微笑」表情
   - 根據不同狀況做出不同反應

4. **計時拍拍樂模式**（進階挑戰） ⏱️
   - 點擊「開始遊戲」按鈕開始 30 秒挑戰
   - 在時間內盡量多拍電子雞
   - 結束後顯示得分

## 📚 學習內容

### 🎯 第一部分：條件判斷（電子雞的大腦）⭐ 核心重點

#### 1.1 什麼是條件判斷？

**生活中的例子**：
- ☀️ 「**如果**天氣好，**那麼**去公園玩」
- 🌧️ 「**如果**下雨，**那麼**在家看書」
- ⏰ 「**如果**很晚了，**那麼**去睡覺，**否則**再玩一下」

**在 Scratch 中**：
```scratch
如果 <條件> 那麼
  做這件事
否則
  做另一件事
結束
```

#### 1.2 第一個條件判斷：根據飢餓度說話

**目標**：讓電子雞根據飢餓度說出不同的話

**程式**：
```scratch
當這個角色被點擊
如果 <(飢餓度) < (30)> 那麼
  造型換成 [傷心]
  說 [我好餓...先餵我吧！] 持續 2 秒
否則
  造型換成 [微笑]
  說 [謝謝你跟我玩！] 持續 2 秒
結束
```

**測試**：
1. 讓飢餓度降到 30 以下 → 點電子雞 → 應該說「我好餓」
2. 餵食讓飢餓度升高 → 點電子雞 → 應該說「謝謝你」

---

### 😊 第二部分：快樂度系統（情緒管理）⭐ 核心重點

#### 2.1 建立快樂度變數

**步驟**：
1. 點擊積木區的「變數」分類
2. 點擊「建立一個變數」
3. 輸入名稱：`快樂度`
4. 選擇「適用於所有角色」
5. 勾選顯示

#### 2.2 快樂度自動下降

**程式**：
```scratch
當 綠旗 被點擊
將 [快樂度] 設為 (80)
重複執行
  等待 (5) 秒
  將 [快樂度] 改變 (-1)
結束
```

**說明**：
- 每 5 秒減少 1 點快樂度
- 模擬電子雞會慢慢變無聊

#### 2.3 點擊增加快樂度

**程式**：
```scratch
當這個角色被點擊
如果 <(快樂度) < (80)> 那麼
  將 [快樂度] 改變 (10)
  造型換成 [微笑]
  說 [和你玩真開心！] 持續 2 秒
  播放音效 [pop]
否則
  說 [我已經很開心了！] 持續 2 秒
結束
```

---

### 👋 第三部分：滑鼠感應（自然互動）⭐ 核心重點

#### 3.1 認識滑鼠偵測積木

**重要積木**：
```scratch
<碰到滑鼠游標？>    ← 檢查滑鼠是否碰到角色
```

#### 3.2 滑鼠接近時的反應

**程式**：
```scratch
當 綠旗 被點擊
重複執行
  如果 <碰到滑鼠游標？> 那麼
    造型換成 [眨眼]
    說 [你要跟我玩嗎？] 持續 1 秒
    等待 (0.5) 秒
  否則
    造型換成 [微笑]
  結束
  等待 (0.1) 秒
結束
```

**效果**：
- 滑鼠接近 → 電子雞眨眼並打招呼
- 滑鼠移開 → 電子雞恢復微笑

{: .note }
**💡 iPad 小提示**：iPad 使用者用手指移動也有同樣效果！

---

### 🧠 第四部分：複合條件判斷（進階邏輯）

#### 4.1 邏輯運算積木

**「且」積木**：兩個條件都要成立
```scratch
<<(飢餓度) < (30)> 且 <(快樂度) < (30)>>
```

**「或」積木**：至少一個條件成立
```scratch
<<(飢餓度) < (20)> 或 <(快樂度) < (20)>>
```

#### 4.2 根據多個條件決定反應

**程式**：
```scratch
當 綠旗 被點擊
重複執行
  如果 <<(飢餓度) < (30)> 且 <(快樂度) < (30)>> 那麼
    造型換成 [傷心]
    說 [我又餓又無聊...] 持續 2 秒
  否則
    如果 <<(飢餓度) > (70)> 且 <(快樂度) > (70)>> 那麼
      造型換成 [微笑]
      說 [我現在超棒！] 持續 2 秒
    否則
      造型換成 [發呆]
      說 [還可以啦] 持續 2 秒
    結束
  結束
  等待 (3) 秒
結束
```

## 🐛 常見問題除錯

### ❌ 問題 1：條件判斷不工作

**可能原因**：
- 變數名稱打錯
- 條件符號用錯（< > =）
- 忘記「結束」積木

**檢查**：

```scratch
  ...

```
