# 📸 第五週必要截圖清單

## 🎯 最小必要版本（6 張）

### 1. 左右移動-鍵盤控制.png ✅（你已完成）
**位置**：電子雞角色的鍵盤控制程式
**內容**：
```
當綠旗被點擊
重複無限次
  如果 <按下 [左移鍵 v] ?> 那麼
    x 改變 (-10)
    如果 <(x 座標) < [-220]> 那麼
      x 設為 [-220]
    結束
  結束
  如果 <按下 [右移鍵 v] ?> 那麼
    ...
```

### 2. 觸控按鈕程式.png
**位置**：左箭頭或右箭頭按鈕的程式
**內容**：
```
當綠旗被點擊
定位到 x:(-200) y:(-150)
重複無限次
  如果 <碰到 [鼠標 v] ?> 那麼
    如果 <滑鼠鍵被按下?> 那麼
      廣播訊息 [向左移動 v]
      等待 (0.1) 秒
    結束
  結束
結束
```

### 3. 蘋果掉落-完整程式.png
**位置**：蘋果角色的「當分身產生」程式
**內容**：
```
當分身產生
定位到 x:(隨機取數(-220)到(220)) y:(180)
顯示
重複無限次
  y 改變 (-5)
  如果 <碰到 [扭蛋雞 v] ?> 那麼
    廣播訊息 [吃到蘋果 v]
    刪除此分身
  結束
  如果 <(y 座標) < [-180]> 那麼
    刪除此分身
  結束
結束
```

### 4. 電子雞接收吃到蘋果.png
**位置**：電子雞角色的接收訊息程式
**內容**：
```
當收到訊息 [吃到蘋果 v]
變數 [分數 v] 改變 [1]
變數 [飢餓度 v] 改變 [15]
變數 [快樂度 v] 改變 [5]
播放音效 [Pop v]
說出 [好吃！] 持續 (0.5) 秒
```

### 5. 建立蘋果分身.png
**位置**：蘋果角色的建立分身程式
**內容**：
```
當綠旗被點擊
重複無限次
  等待 (2) 秒
  建立自己的分身
結束
```

### 6. 隨機數使用範例.png
**位置**：蘋果分身的隨機 X 座標設定
**內容**：
```
定位到 x:(隨機取數(-220)到(220)) y:(180)
```
可以單獨框選「隨機取數」積木放大顯示

## 📝 截圖步驟

1. 開啟「第五週-完成的遊戲.sb3」
2. 點擊對應的角色
3. 找到要截圖的程式區塊
4. 使用 Windows 截圖工具（Win + Shift + S）
5. 框選程式區塊
6. 貼到小畫家或其他圖片編輯器
7. 儲存為對應檔名
8. 放到 `week5/resources/` 資料夾
